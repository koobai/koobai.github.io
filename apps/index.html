Apps.

一款合适的软件，在工作中能成为很好的生产力工具，让工作事半功倍；在生活中能成为很好的消遣工具，让生活充满多姿多彩。以下为日常工作生活中所常用：(2023-01-08 更新)
Sketch
2022年做项目才开始学会使用，画稿利器。虽然现在大家都用Figma，但还是得看环境使用，适合自己才是最好的。
macOS
Pixelmator Pro
处理图片的时候，一直使用Photoshop，在想有没有平替，毕竟也不用那么专业处理。于是发现了这款轻量级软件，整体操作及快捷键跟ps差不多，具体功能还在摸索中～～
macOS
Illustrator 2022
Adobe的矢量软件，平常主要用于素材的修改调整，只会基本的功能。
macOS
Eagle
素材管理软件，支持各类格式。很多年前就已购买，设计师应该人手备一个。
macOS
Mindnode
思维导图软件，单纯因为它的颜值才入手的，使用起来也方便简单。
macOS iOS
Typora
之前一直使用Bear，搭建了Nas之后，想着数据都存本地，这才发现这款软件，界面根据模板自定义之后，用着非常舒服。同类型的软件还有“妙言”，也很不错，但目前还不支持所写所得，希望未来可以更新～～
macOS
1Writer
同样也是Markdown软件，主要通过它的WebDav功能，连接Nas查看Typota所写的记录。
iOS
Surge
上网必备软件，已使用多年。虽然现在同质化的软件很多，但已使用习惯，主要还支持iCloud同步。
macOS iOS
Bob
翻译必备软件，支持各种翻译形式。如果自己申请各大平台翻译密钥，配置好之后，日常翻译非常方便。
macOS
Maccy
剪贴板管理软件，能记录多次的复制内容，方便随时调取使用。免费开源。
macOS
MacZip
压缩软件，已使用多年，免费，是不是开源，忘了。
macOS
MonitorControl
调整显示器亮度软件，适合外接显示器使用。鼠标在哪个显示屏，调整就是哪个显示屏的亮度。免费开源。
macOS
Hidden Bar
隐藏菜单栏图标软件，菜单栏图标过多的时候，使用它非常不错。免费开源。
macOS
Rectangle
调整窗口大小软件，用快捷键方式让窗口根据需要分屏。免费开源。
macOS
Bitwarden
密码管理工具，服务器使用Docker版的vaultwarden，数据存储本地。全平台使用，再也不用记该死的密码了。免费开源。
macOS iOS Web
AppCleaner
卸载app工具，使用了多年，能把软件相关文件都能查找出来，比直接删除干净很多。免费开源。
macOS
Sublime Text
文本编辑软件，搞博客就靠它了，虽然只会用它的皮毛，但能分颜色显示，对我来说就是最大用处，不然密密麻麻的字母真看不清。
macOS
Firefox
一直在使用Google Chrome，最近用了下Firefox，发现打开网站速度似乎比chrome快，那就再继续尝试下～～
macOS iOS
IINA
视频播放软件，并且支持IPTV播放。免费开源。
macOS
Youtube
没什么说的，关注你喜欢的视频频道。
iOS tvOS Web
Infuse
如果你有apple tv，那它就是必备软件。可以让你的视频流以海报的形式展示，并且支持各类格式。
macOS iOS tvOS
Netflix
流媒体视频软件，缺点就是每个月的订阅费确实蛮贵。独自订阅几个月后，在合租平台租了个合租号。
iOS tvOS Web
小宇宙
播客应用，其实很少听博客，但在开车的时候，放着声音很不错。
iOS Web
家庭
家里接入了不少的智能设备，特别是所有的灯具都是智能的。买的是小米系，日常除了使用米家之外，通过Docker版的Home assistant接入HomeKit。如果是苹果的全家桶用户，使用起来非常方便，特别是打开app的速度比米家快了不知道有多少。
macOS iOS
小历
一款日历的组件，使用了很多年，主要用来查看农历跟日常的节假日。
macOS iOS
Telegram
关注你想关注的，你能想到的资源都有。
macOS iOS
彩云天气
一段时间内，特别喜欢收集天气类app，只要看着好看就下载，最终留下了这款，可能因为它数据相对准确，组件也做得不错。
iOS
熊猫吃短信
过滤垃圾信息必备，iOS相关的过滤真是太垃圾了，羡慕隔壁的Android。
iOS
VVebo
第三方微博工具，同类型的还有以前的墨客。不可抗力的原因，目前都已下架。但用了很多年，比官方的好用不少，可惜了～～
iOS
更多。。
